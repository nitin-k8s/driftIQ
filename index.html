Key Features

Pattern Replacement

  • Column-based: Replace values in specific table columns (e.g., mask all port speeds)
  • Regex patterns: Match and replace using regular expressions (e.g., uptime.*)
  • Global or command-specific: Apply rules to all commands or specific ones
  • Persistent rules: Save rules in browser storage for reuse

Smart Filtering

  • Line-level ignores: Click line numbers to ignore specific lines
  • Command ignores: Hide entire commands that don't matter
  • Toggle visibility: Show/hide ignored content on demand
  • Export with filters: Reports maintain your ignore rules

Visual Diff

  • üü¢ Green: Lines that were added
  • üî¥ Red: Lines that were deleted
  • üü° Yellow: Partial changes within lines
  • Strikethrough: Shows exactly what was removed
